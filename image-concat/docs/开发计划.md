# Image Concat 开发计划

## 阶段 1：项目初始化
- 在 `image-concat` 下创建 Vite React 项目并接入 TailwindCSS。
- 完成基础页面骨架与组件拆分。

## 阶段 2：上传与预览
- 完成多图上传、缩略图预览、删除/清空。
- 解析图片原始宽高并建立数据模型。

## 阶段 3：布局引擎
- 实现 Telegram 风格的自动排版（按数量和比例分组布局）。
- 实现随机重排与重新计算。

## 阶段 4：Canvas 合成与导出
- 根据布局结果在 canvas 绘制。
- 实现 PNG/JPG 导出、质量控制、命名规则。

## 阶段 5：参数面板与联动
- 接入间距、圆角、背景、输出倍率等参数。
- 实现“改参数即预览刷新”。

## 阶段 6：体验与稳定性
- 补充异常提示、边界处理（超大图/空数据）。
- 性能优化与移动端适配。
- 自测用例覆盖：2/3/4/6/9 张图片、横竖混合比例。

## 交付里程碑
- M1：可上传并预览多图。
- M2：可自动排版并随机重排。
- M3：可导出高质量拼接图。
- M4：参数可调且移动端可用。

## 验收标准
- 上传 2~9 张图片均可生成拼接结果。
- 布局无明显拉伸变形，留白控制合理。
- 随机重排可重复触发且每次结果可见变化。
- 导出图片与预览一致，主流浏览器可用。
