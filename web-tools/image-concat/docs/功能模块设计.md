# Image Concat 功能模块设计

## 1. 图片输入模块
- 多图上传：支持点击选择/拖拽上传（`jpg/png/webp`）。
- 本地预览：显示缩略图、文件名、原始尺寸、大小。
- 图片管理：删除单张、清空全部、拖拽调整顺序（可选）。

## 2. 拼接布局引擎（核心）
- 自动计算画布尺寸：根据图片数量和比例动态决定行列。
- 智能排列：优先保持图片完整比例，减少留白，接近 Telegram 多图组合风格。
- 间距与圆角：支持统一间距、圆角、背景色。
- 排序策略：
  - 默认顺序（上传顺序）。
  - 随机重排（每次重新打乱）。
  - 固定随机种子（可选，便于复现）。

## 3. 画布渲染模块
- 使用 `canvas` 合成输出图。
- 高质量导出：支持 1x/2x 分辨率。
- 背景渲染：透明/纯色可选。

## 4. 导出下载模块
- 一键导出为单张图片（`png/jpg`）。
- 文件名规则：如 `concat-YYYYMMDD-HHmmss.png`。

## 5. 参数控制面板
- 控制项：间距、圆角、背景色、输出格式、输出质量（jpg）。
- 实时预览：参数变化后即时重算布局并重绘。

## 6. 交互与状态模块
- 状态管理：上传列表、布局参数、生成结果。
- 异常处理：文件类型不支持、超大图片提示、内存占用提示。
- 性能优化：图片懒加载、合成前压缩（可选）。

## 7. 技术方案（Vite + React + TailwindCSS）
- 项目结构建议：
  - `src/components/`：上传区、参数面板、预览区、操作栏。
  - `src/lib/layout/`：布局算法（行列计算、随机重排）。
  - `src/lib/canvas/`：绘制与导出。
  - `src/types/`：图片与布局类型定义。
- 关键实现点：
  - 布局算法输入：图片原始宽高数组 + 目标宽度 + 样式参数。
  - 布局算法输出：每张图在画布中的 `x/y/w/h`。
  - 随机重排：Fisher–Yates 洗牌，保留恢复默认顺序能力。
  - Canvas 导出：`toBlob` 下载，避免大图 `base64` 内存峰值。
