# 定时提醒器 - 设计规格文档

## 概述

一个极简的定时提醒页面，按照固定时间点触发提醒（如每5分钟：19:00、19:05、19:10...）。使用浏览器通知和声音提示。页面加载时自动同步网络时间。

## 视觉设计

- **风格**: 极简、干净
- **布局**: 单卡片垂直居中
- **配色**: 中性灰，激活状态使用微妙强调色

### 布局结构

```
┌─────────────────────────────────┐
│          标准时间 19:05:32       │
├─────────────────────────────────┤
│                                 │
│      1    5    10   15   30     │  ← 预设按钮（分钟）
│                                 │
│   自定义: _________ 分钟         │  ← 弱化的自定义输入
│                                 │
│   提前提醒: [关] 10s 30s 60s    │  ← 提前提醒设置
│                                 │
│        ● 已停止                  │  ← 状态指示
│                                 │
│        [开始]  [停止]            │  ← 操作按钮
│                                 │
│   下次提醒: 19:09:30 (提前)     │  ← 下次触发时间
│            19:10:00 (整点)     │
└─────────────────────────────────┘
```

## 功能

### 1. 网络时间同步
- 页面加载时从 `worldtimeapi.org/api/ip` 获取标准时间
- 在顶部显示同步后的时间，每秒更新
- 获取失败时：降级使用本地时间，并标注 "(本地时间)"
- 失败时可手动重试

### 2. 间隔选择
- **预设按钮**: 1、5、10、15、30 分钟
- **自定义输入**: 手动输入分钟数，视觉上弱化
- 同一时间只能选择一个间隔

### 3. 提前提醒设置
- **预设选项**: 关闭、10秒、30秒、60秒
- **默认状态**: 关闭（不启用提前提醒）
- **作用时间**: 整点触发前 N 秒先提醒一次
- 启用后，每个触发点会有两次提醒：
  - 提前 N 秒提醒一次
  - 整点再提醒一次
- 示例：5分钟间隔 + 30秒提前，当前时间 19:02:00
  - 19:04:30 提前提醒
  - 19:05:00 整点提醒
  - 19:09:30 提前提醒
  - 19:10:00 整点提醒

### 4. 计时器逻辑
- 按固定时间间隔触发（如每5分钟对齐到整刻钟）
- 例如设置5分钟间隔，当前时间 19:02 → 19:05、19:10、19:15... 触发
- 根据当前时间和间隔计算下次触发时间
- 实时显示下次触发时间（区分提前提醒和整点提醒）

### 5. 通知提醒
- 使用浏览器通知 API 弹出提醒
- 首次交互时请求通知权限
- 提前提醒和整点提醒显示不同内容：
  - 提前提醒："提前提醒 - 19:10:00 即将到来"
  - 整点提醒："19:10:00"

### 6. 声音提示
- 使用 Web Audio API 生成柔和铃声
- 约2秒的轻柔钟声音效
- 提前提醒和整点提醒使用相同声音
- 与通知同时播放

### 7. 状态
- **空闲**: 计时器未运行，显示"已停止"
- **运行中**: 计时器激活，显示"计时中"带脉冲动画

## 组件清单

### 标准时间显示
- 以 HH:MM:SS 格式显示当前时间
- 前缀标签 "标准时间"
- 网络失败时显示 "(本地时间)"

### 预设按钮
- 一排5个按钮：1、5、10、15、30
- 激活状态：高亮背景色
- 悬停：微妙的颜色变化

### 自定义间隔输入
- 带"分钟"标签的文本输入框
- 视觉上弱化（文字颜色较浅、字号较小）
- 回车或失焦时触发

### 提前提醒设置
- 一排4个按钮：关闭、10s、30s、60s
- 默认选中"关闭"
- 视觉风格与预设间隔按钮一致
- 启用时按钮高亮

### 状态指示器
- 简单文字指示
- 空闲："○ 已停止"
- 运行中："● 计时中" 带 CSS 动画

### 操作按钮
- "开始" 按钮 - 以选定间隔启动计时器
- "停止" 按钮 - 停止计时器
- 未选择间隔时禁用

### 下次提醒显示
- 以 HH:MM 格式显示下次触发时间
- 随着倒计时实时更新

## 技术实现

### 文件结构
```
web-tools/interval-timer/
├── index.html    # 单文件，内嵌 CSS/JS
└── SPEC.md       # 本文档
```

### 外部依赖
- 无（纯原生 HTML/CSS/JS）

### 时间 API
- 端点: `https://worldtimeapi.org/api/ip`
- 降级方案: `Date()` 本地时间

### 声音生成
- Web Audio API
- OscillatorNode 使用正弦波
- ADSR 包络实现柔和的起音/衰减
- 频率: ~440Hz 基频带谐波

### 浏览器兼容性
- 支持通知 API 的现代浏览器
- 优雅降级: 拒绝通知时仅使用声音

## 验收标准

1. 页面加载后立即同步网络时间
2. 时间显示每秒更新
3. 选择预设按钮后高亮显示并启用开始按钮
4. 自定义输入接受数字分钟数
5. 提前提醒按钮可选中，默认关闭
6. 开始按钮以选定间隔启动计时器
7. 计时器在正确的整点分钟触发（不是倒计时）
8. 启用提前提醒时，提前 N 秒先触发一次，整点再触发一次
9. 提前提醒和整点提醒的通知内容不同
10. 每次触发时播放声音
11. 停止按钮终止计时器
12. 网络失败时显示本地时间并标注指示
