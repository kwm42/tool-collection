# Prompt Generator - 需求分析文档

## 1. 项目概述

**项目名称**: Prompt Generator  
**项目类型**: ComfyUI 人物图像生成提示词工具  
**核心功能**: 一键随机生成人物图像提示词，支持预设模板和自定义组合  
**目标用户**: 使用 ComfyUI 进行 AI 人物图像生成的用户

---

## 2. 核心功能需求

### 2.1 功能列表

| 功能模块 | 描述 | 优先级 |
|---------|------|--------|
| 一键随机生成 | 随机所有维度，生成完整提示词 | **P0** |
| 维度随机 | 单维度随机切换 | P0 |
| 维度锁定 | 锁定维度，避免被随机改变 | P0 |
| 预览输出 | 展示当前配置的提示词 | P0 |
| 一键复制 | 复制提示词到剪贴板 | P0 |
| 历史记录 | 保存最近生成的组合 | P0 |
| 预设模板 | 固定风格模板（可选） | P1 |
| 手动微调 | 点击展开抽屉，选择预设或自定义组合 | P1 |

### 2.2 设计原则

1. **随机即核心**: 一键生成是用户第一操作，界面突出展示
2. **所见即所得**: 每个维度显示当前值，无需展开就知道状态
3. **锁定保护**: 锁定后随机不会改变该维度
4. **快速切换**: 点击选项立即生效，无需确认
5. **历史追溯**: 生成的组合自动保存，可快速复用

## 5. 非功能性需求

### 5.1 性能要求
- 首屏加载 < 1s
- 配置切换 < 50ms
- 支持移动端适配

### 5.2 可用性
- 多选支持（同一维度可选多个标签）
- 键盘快捷键操作
- 本地存储用户配置
