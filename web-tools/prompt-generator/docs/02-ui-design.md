# Prompt Generator - UI 设计文档

## 1. 设计原则

- **随机即核心**: 随机生成是第一入口，界面最醒目位置
- **简化手动**: 手动选择作为辅助，层级次要
- **锁定保护**: 锁定维度保留个人偏好，减少重复配置
- **快速迭代**: 一键随机 + 复制，快速生成提示词

## 14. 界面交互设计

### 14.1 核心交互流程

```
┌─────────────────────────────────────────────┐
│                                             │
│      [ 🎲 一键随机生成 ]    ← 点击          │
│                                             │
│              ↓                               │
│                                             │
│   生成结果 + 自动复制提示 + 加入历史          │
│                                             │
│              ↓                               │
│                                             │
│   可选择：锁定维度 / 手动微调 / 继续随机      │
│                                             │
└─────────────────────────────────────────────┘
```

### 14.2 一键生成按钮

- **尺寸**: 大尺寸主按钮，最醒目
- **位置**: 页面顶部，居中
- **文字**: "🎲 一键随机生成"
- **点击效果**: 随机所有未锁定维度，生成提示词
- **悬停**: 主色悬停背景，轻微放大
- **点击反馈**: scale 缩小 + 骰子旋转动画

### 14.3 维度行（简化设计）

随机为主，展开为辅：

```
┌─────────────────────────────────────────────────────────┐
│ [图标] [维度名]  当前值...        [🔒] [🔀]           │
└─────────────────────────────────────────────────────────┘
  ↑                ↑       ↑       ↑
  次要           次要    锁定    随机
                (点击可展开)
```

| 元素 | 优先级 | 交互 |
|------|--------|------|
| 维度图标 + 名称 | 次要 | 辅助识别 |
| 当前值 | 可见 | 告知当前状态 |
| 锁定按钮 | 常用 | 点击切换锁定 |
| 随机按钮 | 常用 | 点击随机该维度 |
| 展开详情 | 次要 | 点击展开抽屉（手动微调用） |

**简化说明:**
- 移除单独的"展开"按钮，改为点击整行或当前值区域展开
- 维度行保持简洁，不显示过多选项

### 14.4 抽屉面板（辅助工具）

作为手动微调的辅助入口：

| 设计要点 | 说明 |
|----------|------|
| 入口隐藏 | 需点击维度行展开，非默认展示 |
| 预设优先 | 默认显示预设模式 |
| 自定义次要 | 自定义组合折叠收起 |
| 快速确认 | 选中预设后一键确认 |

**抽屉结构（简化）:**
```
┌─────────────────────────────────────┐
│ [标题]                        [×]   │
├─────────────────────────────────────┤
│ 预设模式:                          │
│ ┌─────────┐ ┌─────────┐ ┌───────┐ │
│ │  预设1  │ │  预设2  │ │ 🎲    │ │
│ └─────────┘ └─────────┘ └───────┘ │
│                                     │
│ [展开自定义组合 ▼] ← 默认折叠       │
└─────────────────────────────────────┘
```

### 14.5 预览面板

```
┌─────────────────────────────────────────┐
│ 1girl, beautiful face, long hair...     │
│                              [复制] 🔄  │
└─────────────────────────────────────────┘
                    ↑           ↑
                  最重要      常用
```

| 按钮 | 优先级 | 说明 |
|------|--------|------|
| 复制 | P0 | 一键复制到剪贴板 |
| 刷新 | P1 | 重新随机当前配置 |

### 14.6 历史记录（可折叠）

```
┌─────────────────────────────────────┐
│ 历史记录 [2] [▼]                  │  ← 可折叠
└─────────────────────────────────────┘
        ↓ 点击展开
┌─────────────────────────────────────┐
│ 历史记录 [2] [▲]                  │
├─────────────────────────────────────┤
│ #1 瓜子脸+长发 | 学院风 | 夕阳     │
│    [应用] [复制] [删除]            │
│ #2 圆脸+双马尾 | 运动服 | 夜晚    │
│    [应用] [复制] [删除]            │
└─────────────────────────────────────┘
```

### 14.8 快捷操作

| 操作 | 位置 | 效果 |
|------|------|------|
| 一键随机 | 页面顶部 | 随机所有维度 |
| 单维度随机 | 维度行 | 随机该维度 |
| 锁定/解锁 | 维度行 | 保护维度不被随机 |
| 复制结果 | 预览面板 | 复制提示词 |
| 重新随机 | 预览面板 | 保持锁定重新随机 |

## 15. 交互状态汇总

### 15.1 组件状态

| 组件 | 状态 | 样式变化 |
|------|------|---------|
| 生成按钮 | 默认/悬停/点击/禁用 | 背景色/主色悬停/scale/透明度 |
| 维度行 | 默认/悬停/锁定 | 背景/悬停背景/锁定背景 |
| 维度行(当前值) | 默认/悬停 | 次要文字/主文字，点击展开抽屉 |
| 锁定按钮 | 默认/锁定 | 次要文字色/红色 |
| 随机按钮 | 默认/悬停/点击 | 随机色，悬停放大，点击旋转 |
| 预设卡片 | 默认/悬停/选中 | 边框/主色边框/选中背景 |
| 选项标签 | 未选中/选中/悬停 | 灰背景/主色背景/浅蓝背景 |
| 输入框 | 默认/聚焦/禁用 | 边框/主色边框/灰色背景 |

### 15.2 动画效果

| 场景 | 动画 |
|------|------|
| 一键生成点击 | 骰子旋转 + 结果渐显，500ms |
| 抽屉滑入 | 右侧滑入，300ms，spring 缓动 |
| 抽屉滑出 | 右侧滑出，300ms |
| 单维度随机 | 图标旋转一圈，300ms |
| 锁定切换 | 图标缩放，200ms |
| 按钮悬停 | scale(1.02)，150ms |
| 按钮点击 | scale(0.95)，150ms |
| 复制成功 | 按钮短暂变绿 + 文字变为"已复制"，1s 后恢复 |

### 15.3 功能优先级总结

| 功能 | 优先级 | 展示方式 |
|------|--------|---------|
| 一键随机生成 | P0 | 页面最醒目位置，大按钮 |
| 复制结果 | P0 | 预览面板内，一键复制 |
| 单维度随机 | P0 | 维度行显眼按钮 |
| 锁定维度 | P0 | 维度行按钮 |
| 重新随机 | P1 | 预览面板 |
| 手动微调 | P2 | 点击展开抽屉 |
| 历史记录 | P2 | 可折叠面板 |
