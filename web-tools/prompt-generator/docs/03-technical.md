## 3. 数据结构

### 3.1 预设文件

每个维度对应一个 JSON 文件：

| 文件 | 对应维度 |
|------|---------|
| appearance.json | 外貌 |
| hairstyle.json | 发型 |
| body.json | 身材 |
| action.json | 行为 |
| clothing.json | 服装 |
| time.json | 时间 |
| scene.json | 场景 |
| weather.json | 天气 |
| lighting.json | 光照 |
| composition.json | 构图 |
| props.json | 道具 |
| quality.json | 质量 |

### 3.2 预设文件内容

预设文件包含两个部分：

| 部分 | 说明 |
|------|------|
| 预设列表 | 完整的提示词组合预设 |
| 子维度选项 | 可自由组合的细分选项 |

预设支持权重配置，影响随机选择概率。

### 3.3 JSON 结构示例

```json
{
  "dimension": "维度标识",
  "label": "中文名称",
  "icon": "图标",
  "presets": [
    {
      "id": "唯一标识",
      "name": "预设名称",
      "prompt": "英文提示词",
      "weight": 1.0
    }
  ],
  "subDimensionOptions": {
    "子维度名1": [
      { "value": "选项值", "prompt": "提示词", "weight": 1.0 }
    ],
    "子维度名2": [
      { "value": "选项值", "prompt": "提示词", "weight": 1.0 }
    ]
  }
}
```

**字段说明：**

| 字段 | 类型 | 说明 |
|------|------|------|
| dimension | string | 维度唯一标识 |
| label | string | 界面显示的中文名称 |
| icon | string | 界面显示的图标 |
| presets | array | 预设列表 |
| presets[].id | string | 预设唯一ID |
| presets[].name | string | 预设显示名称 |
| presets[].prompt | string | 预设对应的英文提示词 |
| presets[].weight | number | 随机权重，影响选中概率 |
| subDimensionOptions | object | 子维度选项分组 |
| subDimensionOptions[].value | string | 选项显示值 |
| subDimensionOptions[].prompt | string | 选项对应的英文提示词 |
| subDimensionOptions[].weight | number | 随机权重 |

## 4. 状态设计

### 4.1 状态类型

| 状态类型 | 说明 |
|----------|------|
| 维度状态 | 各维度的锁定状态、当前选择 |
| 历史状态 | 生成的记录列表 |
| UI状态 | 抽屉展开、加载等 |

### 4.2 状态流转

```
初始状态 → 点击生成 → 随机选择 → 更新状态 → 保存历史 → 展示结果
```

---

## 5. 核心逻辑

### 5.1 随机算法

- 加权随机：根据预设权重决定选择概率
- 单维度随机：独立于其他维度
- 全局随机：遍历所有未锁定维度

### 5.2 提示词生成

按固定顺序拼接各维度的提示词，最后添加负面提示词。

| 任务 | 说明 |
